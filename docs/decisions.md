## ① MVP 컷 — 확정

[architecture-v2.md](./architecture-v2.md) 풀스코프 → MVP 컷 그대로 진행.

**유지 (MVP):**
- Auth: 이메일+패스워드 가입/로그인/세션
- Library: 시드 모듈 3개 (아래 ② 참조)
- Practice: 텍스트 레이어 + 캔버스 단일 모드(자유형) + 도구 최소
- AI Tutor: 질문/응답/히스토리 + 응답 별점(1–5)
- Session: 세션 저장 + 자가평가 1줄
- Admin: 가입 승인 1화면
- 분석: 인지/가치/시간 3축 *기록*만(점수 컬럼 채움, 리포트 미생성)

**컷 (다음 스프린트):**
이메일 인증·소셜·비번찾기·온보딩·결제·캔버스 제약/단계 모드·갤러리·다운로드·알림·리마인더·성장리포트·3축 리포트·관리자 확장(콘텐츠/모듈/통계/그룹/AI품질/프롬프트 패널)·백그라운드 잡·운영 보조 에이전트.

QA는 E2E 3종 대신 **스모크 1종**: 가입 → 첫 학습 → 캔버스 1회 → 튜터 1턴 → 자가평가 → 세션 저장.
