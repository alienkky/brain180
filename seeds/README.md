## 검수 체크리스트

각 파일의 다음 4가지를 기영님이 확인:

1. **text_body** — 인용 범위 / 번역 판본 OK?
2. **axis_focus** — 인지/가치/시간 어느 축이 dominant?
3. **target_nodes** — 학습자가 추출해야 할 개념 노드 4~8개
4. **trap_concepts** — 표면적 독해가 빠지기 쉬운 함정

수정사항은 같은 파일에 직접 편집 → push 후 ALI-66·ALI-62가 받음.
